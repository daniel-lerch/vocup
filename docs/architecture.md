# Vocup technical architecture

## UI Framework

### Considered frameworks

#### Windows Forms
➕ Easy to use
➖ No data binding
➖ Not styleable
➖ Limited to Windows

#### Avalonia
➕ Cross platform
➕ Clean API
➖ Not mature 
➖ No mobile apps

#### Blazor Hybrid
➕ Runs everywhere
➕ Partial modernization
➖ Complex interop with component libraries
➖ Data binding is buggy

#### WinUI 3
➕ Recommended by Microsoft
➖ Experimental
➖ No unified API for UWP and Win32
➖ No mobile apps
